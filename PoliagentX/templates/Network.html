Network of Interdependencies

One of the model's inputs is a network of conditional dependencies between all the indicators in the data. Upload a file containing a matrix that captures such network.

The template provided in this step is organised in three sheets. The first sheet (labelled template_network) is where you must put the information about the interdependencies. The second sheet (labelled metadata) explains each column in the template. The third sheet (labelled example_network) provides an example network corresponding to the example synthetic data from the other two templates.

Don't have data? Use the synthetic examples provided in the template's "example_network" tab for experimentation.
